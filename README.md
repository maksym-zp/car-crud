# Laravel && Vue

* Laravel 6.14.0

            
##Installation

### Laravel
```sh
$ git clone
-----
update the .env file database connection
-----
$ composer install && composer update
$ php artisan key:generate 
$ php artisan migrate --seed
$ php artisan storage:link
$ npm install

```

### Run

```sh
$ php artisan serve
$ npm run dev    

---
in developing ...

```
